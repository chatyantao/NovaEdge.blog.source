# 一、前期准备：源图质量决定 80% 视频效果

AI 视频的完成度，很大程度取决于你给的“源图”。

## 源图黄金标准（务必遵守）

- 比例：16:9 横屏

# 四、从故事到视频的完整工作流（高阶）

适合做系列短剧、IP 视频、YouTube 故事类频道。

## 工作流步骤

1. **生成微故事**  
   ChatGPT/Claude 生成 300–500 字短篇故事。

2. **角色设定**  
   用结构化描述锁定角色外观 + 性格。

3. **分镜生成**  
   让模型按照“Pixar Storyboard Prompt”生成每一帧画面提示词。

4. **图生视频**  
   将每张分镜丢入 Img2Video 工具：Sora / Veo / Kling / Runway。

5. **剪辑整理**  
   导出后做轻量剪辑（衔接 / 色彩 / 音频）。

# 五、常见问题避坑总结

## 1. 人脸崩坏、换脸

- 使用近景清晰源图  
- 明确指定：  
ensure the face is consistent with the reference

复制代码
- 避免多人入镜

## 2. 口型错误

- 英文最稳  
- 中文拆分短句  
- 必要时用拼音输入

## 3. 运镜抖动

- 提示词加入：stable camera  
- 动作越复杂 → 镜头越简单  
- 镜头越复杂 → 动作越简单

# 七、音效脚本结构（Video + Audio = 完整体验）

优秀的视频需要三层声音：

### 1. 环境音（持续氛围）

- 风声、海浪、城市噪音  
- 给画面带来空间感

### 2. 动作音（关键声效）

- 脚步、爆炸、衣物摩擦、机械声  

### 3. 情绪音乐（Mood Music）

- 钢琴、合成器、管弦乐等  
- 决定整条视频的情绪色彩
